Este es un párrafo de ejemplo para repasar la estructura básica de un documento HTML.

La etiqueta <p> se utiliza para definir párrafos en HTML.

Uso de la etiqueta <h1> nos sirve para los titulos grandes

El uso de la etiqueta <h2> nos sirve para los titulos medianos

El uso de la etiqueta <h3> nos sirve para los titulos pequeños

El uso de la etiqueta <h4> nos sirve para los titulos más pequeños

El uso de la etiqueta <h5> nos sirve para los titulos aún más pequeños
El uso de la etiqueta <h6> nos sirve para los titulos más pequeños de todos

Ejemplo del uso con listas

Listas

La etiqueta ul nos permite crear una lista desordenada por puntos

    Mientras que la etiqueta <li> nos permite listar los elementos dentro de la etiqueta <ul>

  • ASUS
  • ACER
  • MACBOOK
  • DELL

ahora si usamos la etiqueta <ol> nos sirve para listas ordenadas

  1. ASUS
  2. ACER
  3. MACBOOK
  4. DELL

Para insertar imagenes usamos la etiqueta <img>

src nos sirve para indicar la ruta de la imagen

alt nos sirve para describir la imagen

width y height nos sirven para definir el tamaño de la imagen

title nos sirve para agregar un título a la imagen
