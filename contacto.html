Tablas

La etiqueta <table> se utiliza para crear tablas en HTML.

El elemento border se utiliza para agregar un borde a la tabla.

La etiqueta <tr> se utiliza para definir una fila en una tabla.

La etiqueta <th> se utiliza para definir una celda de encabezado en una tabla.

Formularios

La etiqueta <form> se utiliza para crear formularios en HTML.

El atributo action define a dónde se envían los datos del formulario.

El atributo method define el método de envío de los datos (GET o POST).

Elementos dentro de lformulario

la etiqueta <label> se utiliza para definir etiquetas para los campos del formulario.

El atributo for se utiliza para asociar una etiqueta con un campo de entrada específico.

Campos de entrada

la etiqueta <input> se utiliza para crear campos de entrada en el formulario.

El atributo type define el tipo de campo de entrada (texto, correo electrónico, etc.).

El atributo id define un identificador único para el campo de entrada.

El atributo name define el nombre del campo de entrada, que se utiliza para identificar los datos enviados.





Listas desplegables

La etiqueta <select> se utiliza para crear listas desplegables en el formulario.

La etiqueta <option> se utiliza para definir las opciones dentro de una lista desplegable.

el atributo value se utiliza para especificar el valor que se enviará cuando se seleccione una opción.
